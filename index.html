Diapositiva 6: El socket

Título: 3. El socket

Contenido:

  • El socket es el lugar donde se aloja el microprocesador:
  • Puntos guía: permiten colocar correctamente el microprocesador.
  • Horquilla: permite fijar el microprocesador al socket.
  • Cada socket es específico de un microprocesador aunque puede albergar diferentes microprocesadores.
  • Fabricantes más comunes: Intel y AMD.
